# Applications

Outils, scripts, automatisations et dashboards produits pour toi.

Chaque projet dans son propre sous-dossier, nommé selon la convention : `AAAA-MM-NOM-COURT`

Exemples :
- `2026-06-scraper-linkedin/` — script d'extraction de données LinkedIn
- `2026-06-dashboard-ca/` — tableau de bord suivi chiffre d'affaires
- `2026-07-automatisation-factures/` — workflow de génération de factures
